# everlabTask

## To run the backend

- Go to backend directory
- run `npm i` & `npx prisma generate`
- Run docker `docker compose up -d`
- Go to adminer to `localhost:8080` and do import and import the `db-dump.sql` file
- run `npm run dev`

## To run the frontend

- Go to the frontend directory
- run `npm run dev`
